Chatbot API

Chatbot API'si, kullanıcıdan alınan metne karşılık gelen cevabı döndürmek için kullanılabilir.

API Endpoint

/api

Method

POST

Request Format

{
  "msg": "Kullanıcının sorduğu soru"
}

Response Format

{
  "response": "Chatbot'un verdiği cevap"
}

Örnek Kullanım

import requests

data = {
  "msg": "Merhaba Chatbot!"
}

response = requests.post("https://localhost:5000/api", json=data)
print(response.json()["response"])
